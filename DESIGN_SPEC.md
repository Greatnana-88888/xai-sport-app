## 3. 间距系统

### 3.1 基础间距
| 间距名称 | 大小(px) | 使用场景 |
| :--- | :--- | :--- |
| xs | 4 | 元素内微小间距 |
| sm | 8 | 元素内间距 |
| md | 16 | 卡片内间距、组件间距 |
| lg | 24 | 模块间距 |
| xl | 32 | 大模块间距 |
| xxl | 48 | 页面级间距 |

### 3.2 布局间距
- **内容边距**: 24px (左右)
- **卡片间距**: 16px (卡片之间)
- **段落间距**: 16px (正文段落)

## 4. 卡片布局规范

### 4.1 卡片基础样式
- **圆角**: 32px (标准卡片)
- **阴影**: 0 4px 12px rgba(0, 0, 0, 0.08)
- **边框**: 1px solid #E8E8E8
- **内边距**: 20px

### 4.2 卡片圆角规范
| 卡片类型 | 圆角值(px) | Tailwind类名 | 使用场景 |
| :--- | :--- | :--- | :--- |
| 超大卡片 | 40 | rounded-2xl | 运动30分钟英雄区 |
| 标准卡片 | 32 | rounded-card | 三分化训练、定制计划、开始训练 |
| 小卡片 | 24 | rounded-feature | AI动态建议、好友动态 |

### 4.3 卡片类型

#### 4.3.1 信息卡片
```
宽度: 320px (移动端) / 360px (桌面端)
高度: 根据内容自适应
圆角: 32px
背景: #FFFFFF
```

#### 4.2.2 数据卡片
```
宽度: 100% (单列) / 50% (双列) / 33.33% (三列)
高度: 120px - 180px
圆角: 16px
背景: 渐变背景或纯色
```

#### 4.2.3 功能卡片
```
宽度: 100%
高度: 80px
圆角: 12px
背景: #FFFFFF
边框: 1px solid #E8E8E8
```

## 7. 布局网格

### 7.1 移动端 (≤768px)
- **列数**: 1列
- **边距**: 16px (左右)
- **卡片宽度**: 100%

### 7.2 平板端 (768px - 1024px)
- **列数**: 2列
- **边距**: 24px (左右)
- **卡片宽度**: 50%

### 7.3 桌面端 (≥1024px)
- **列数**: 3列
- **边距**: 48px (左右)
- **卡片宽度**: 33.33%

## 8. 阴影规范

| 阴影等级 | CSS值 | 使用场景 |
| :--- | :--- | :--- |
| 无 | none | 普通卡片 |
| 低 | 0 2px 8px rgba(0, 0, 0, 0.04) | 悬浮元素 |
| 中 | 0 4px 12px rgba(0, 0, 0, 0.08) | 卡片、弹窗 |
| 高 | 0 8px 24px rgba(0, 0, 0, 0.12) | 模态框、下拉菜单 |

## 9. 状态规范

### 9.1 加载状态
- 使用骨架屏或加载动画
- 动画时长: 1.5s

### 9.2 空状态
- 显示图标 + 提示文字
- 文字颜色: #BBBFC4

### 9.3 错误状态
- 显示错误图标 + 错误提示
- 可提供重试按钮

## 10. 动效规范

### 10.1 过渡动画
- 过渡时间: 0.3s
- 缓动函数: ease-out

### 10.2 常见动效
- 悬停: 轻微缩放(1.02) + 阴影加深
- 点击: 轻微缩放(0.98)
- 页面切换: 淡入淡出
- 列表滚动: 惯性滚动
